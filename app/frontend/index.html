About

How it works

StayFinder is a RAG (Retrieval-Augmented Generation) pipeline built on top of a PostgreSQL 15 relational database. User queries are embedded using sentence-transformers, matched against hotel vectors stored in Pinecone, and the top candidates are passed to an LLM for a contextualised natural-language recommendation.

The relational layer handles aggregations, joins, subqueries, CTEs, and window functions — the SQL queries powering the Analytics tab are part of the Milestone 1 deliverable for Database Management Systems at IIT Madras Zanzibar.
